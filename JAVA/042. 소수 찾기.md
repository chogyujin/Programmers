처음엔 그냥 무작정 풀다가 시간초과가 났다.  
아리토스테네스의 체의 개념을 알아야만 풀 수 있는 문제.  
  
```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        boolean[] prim = new boolean[n+1];
        for(int i=2;i<=n;i++){
            if(!prim[i]){
                answer++;
                for(int j=1;i*j<=n;j++){
                    prim[i*j]=true;
                }
            }
        }
        
        return answer;
    }
}
```
